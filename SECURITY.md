# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >= 10.x | :white_check_mark: |
| < 10.0  | :x:                |

## Reporting a Vulnerability

⚠️ **DO NOT** open public GitHub issues for security vulnerabilities! ⚠️

If you believe you've found a security-related bug, fill out a vulnerability report. To do so, follow these instructions:

- Click on the `Security` tab at the top of the repository
- Click `Report a vulnerability` button at the top right corner
- Fill in the form as accurately as you can, including as many details as possible
- Click the `Submit report` button at the bottom
